# UI Component Classification and Visual Layout Analysis for Webpages

Machine learning pipeline for detecting UI components in webpage snapshots using YOLOv11s. Evaluating detected components with standard mathematical approach for clutter, alignment and color contrast.

## Setup

### 1. Clone and Install

```bash
git clone <repository-url>
cd component-classifier-backend/models/
pip install -r requirements.txt
```

### 2. Set Python Path

**Important:** Set PYTHONPATH to ensure imports work correctly.

#### Windows (PowerShell)

```powershell
cd component-classifier-backend/Python/
$env:PYTHONPATH = $PWD

# Verify
echo $env:PYTHONPATH
```

#### macOS / Linux (Bash)

```bash
cd component-classifier-backend/Python/
export PYTHONPATH=$PWD

# Verify
echo $PYTHONPATH
```

**Expected output:** You should see the full path to your `models/` directory.

> **Note:** This must be set each time you open a new terminal. To make it permanent, add the export/set command to your shell profile (`.bashrc`, `.zshrc`, or PowerShell profile).

---

### 3. Prepare Dataset

```bash
mkdir -p data/raw_data/
```

Download webui-70k dataset and extract into `data/raw_data/`

### 4. Configure

```bash
cp config/config_example.yml config/config.yml
```

Edit `config.yml` with your settings

---

## Run Pipeline

### Step 1: Directory Cleanup

```bash
python data_preprocessing\directory_cleanup.py
```

**Output:** Unwanted files removed from `data/raw_data/`

---

### Step 2: Initialize Resource

```bash
python data_preprocessing\resource_initializer.py
```

**Output:** Images moved to `datasets/images/`, `.gz` files extracted

---

### Step 3: Preprocess Data

```bash
python data_preprocessing\json_processor.py
```

**Output:** YOLO labels created in `datasets/labels/`

---

### Step 4: Split Data

```bash
python training\data_splitter.py
```

**Output:** `split_data/train/`, `split_data/validation/`, `split_data/test/`

---

### Step 5: Train Model

```bash
python training\train.py
```

**Output:** `checkpoints/train/weights/best.pt`

---

## GPU Configuration

Edit `training/config.py` based on your GPU:

### RTX 4090 (24GB VRAM)

```python
TRAINING_CONFIG = {
    "batch": 16,
    "workers": 8,
    "device": 0,
}
```

### RTX 2060 (6GB VRAM)

```python
TRAINING_CONFIG = {
    "batch": 4,
    "workers": 4,
    "device": 0,
}
```

---

## Resume Training

Edit `training/config.py`:

```python
TRAINING_CONFIG = {
    "model": "checkpoints/train/weights/last.pt",
    "epochs": 200,
}
```

Then run:

```bash
python training\train.py
```

## Start Fresh

```bash
rm -rf checkpoints/
rm -rf split_data/
rm data.yaml
```

Edit `training/config.py`:

```python
TRAINING_CONFIG = {
    "model": "yolo11s.pt",
}
```
